## PT(Personal Trainer) 관리
 PT 이용에 관련된 정보(PT 가격, 리뷰 작성, 리뷰 사진 등록, 도움이 돼요/안돼요 등..) 를 관리합니다.

## API 상세 설명 
 
### 리뷰 등록
 PT에 대한 리뷰를 등록합니다.(리뷰 내용, 별점, 리뷰 사진 등..)
 <br><br>
 
  
> #### Request
 |메서드|요청 URL|요청 Data|
|----|------|--------------|
|POST|http://localhost:8080/api/v1/pt/pt-id/review||
